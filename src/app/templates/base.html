🎉 Sqlfluff Online 🎉

Fluff is a SQL formatter that I like a lot. I built this app because I often want to format one-off, adhoc sql that isn't even worth saving to a file. You can check out the source code for this application on GitHub (link).

The premise is simple: paste your SQL and select your dialect. Hit that button and find your fixed SQL with the associated linter errors. Enjoy, friends 🌝.
